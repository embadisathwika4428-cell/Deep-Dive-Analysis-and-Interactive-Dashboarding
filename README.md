# Deep-Dive-Analysis-and-Interactive-Dashboarding
This project presents a deep-dive analysis of the Titanic dataset using Google Looker Studio. It includes key performance indicators (KPIs), interactive visualizations, survival analysis based on gender and passenger class, and insights derived through dashboarding techniques for data-driven decision making.
# Task 3 - Deep Dive Analysis & Interactive Dashboarding

## Objective
Analyze Titanic passenger survival patterns using an interactive dashboard.

## KPIs
- Total Passengers
- Total Survivors
- Survival Rate
- Average Age
- Female Survival Count

## Deep Dive Analysis
Segmentation Analysis based on:
- Gender
- Passenger Class
- Age

## Key Findings
- Female passengers showed higher survival rates.
- First-class passengers had better survival chances.
- Third-class passengers had lower survival rates.

## Tools Used
- Google Looker Studio
- CSV Dataset
- GitHub

## Dashboard
Interactive dashboard created using Looker Studio.
